JavScript Intro

JavaScript is a high-level object-oriented multi-paradigm programming language. Programming language is a tool that allows us to write code that will instruct computer to do something. Js is a high level lang that means we don't have to worry about complex stuff like memory management. Object oriented implies lang is mostly based on concept of objects for storing most kinds of data. Multi-paradigm means it is so flexible and versatile that we can use all kinds of different programming styles

Role of JS in web development

Html is responsible for the content of the page. Css is responsible for the presentation of the content. ie styling and laying out elements on a webpage. JS adds dynamic and interactive effects to any webpage. Html is the noun (eg: p tag for paragraph), css is the adjective (paragraph is red), Js is the verb (paragraph is hidden)

React, Angular, Vue are tools that make writing modern, large scale web applications, a lot easier and faster. All these frameworks and libraries are still 100% based on JavaScript. JavaScript versions starting from ES2015/ES6 is called modern javascript. ES => ECMAScript
